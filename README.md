React app built with Vite to search named CSS colors. Used in my seminar fundamentals of digital communicaiton at university of arts Braunschweig.

## Installation

```bash
npm install
```

## Usage

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

```bash
upload dist folder somewhere
```
